# CASIO PRO Calculator

A modern scientific calculator inspired by CASIO calculators, built using HTML, CSS, and JavaScript.

## Features

- Basic arithmetic
- Scientific functions
- DEG/RAD mode
- Trigonometric & inverse trigonometric functions
- Logarithm & natural logarithm
- Square root
- Power functions (x², x³, xʸ)
- Factorial
- Memory functions (MC, MR, M+, M-)
- ANS recall
- Calculation history
- Keyboard support
- Glassmorphism UI
- Ripple button animations

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
