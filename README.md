# CASIO PRO Scientific Calculator

A modern scientific calculator inspired by CASIO calculators, built using HTML, CSS and JavaScript.

## Features

- Basic arithmetic
- Scientific functions
- Trigonometric functions
- Inverse trigonometric functions
- DEG/RAD mode
- Logarithm (log)
- Natural logarithm (ln)
- Square root
- Power functions (x², x³, xʸ)
- Factorial
- Memory (MC, MR, M+, M-)
- ANS recall
- History panel
- Keyboard support
- Glassmorphism UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
